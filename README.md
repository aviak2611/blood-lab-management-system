# 🩸 Blood Lab Management System

## 📌 Overview

A web-based system to manage blood test bookings, reports, employees, and billing.

## 🚀 Features

* User Registration & Login
* Book Blood Tests
* Admin Dashboard
* Employee Management
* Reports & Billing System

## 🛠️ Tech Stack

* PHP
* MySQL
* HTML, CSS

## ▶️ How to Run

1. Install XAMPP
2. Move project to htdocs
3. Import `bloodtest.sql`
4. Run on http://localhost
